# ruijie-go
哈尔滨工业大学威海校园网，锐捷认证跨平台登录脚本

## 使用说明

下载：https://github.com/VaalaCat/ruijie-go/releases

win:
```bash
.\ruijie-go.exe 学号 密码
```

linux:
```
./ruijie-go 学号 密码
```

ios版快捷指令(支持siri和自动化)链接：https://url.vaala.cloud/7okaeR

请在编辑中修改自己的学号姓名
